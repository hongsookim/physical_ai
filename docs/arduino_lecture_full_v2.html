Code
💡
기본 LED 깜빡이기
// 처음 한 번만 실행 void setup() { pinMode(13, OUTPUT); // 13번 핀 출력 설정 } // 계속 반복! (무한 루프) void loop() { digitalWrite(13, HIGH); // LED 켜기 delay(1000); // 1초 기다리기 digitalWrite(13, LOW); // LED 끄기 delay(1000); }
💡 쉽게 이해하기

setup() = 학교 가기 전 준비 (한 번만!)
loop() = 수업 시간표 (계속 반복!)
delay(1000) = 1초 기다리기 (1000ms = 1초)

⚙️
Claude Code가 쓰는 CLI 명령어
# 아두이노 포트 찾기 arduino-cli board list # 코드 컴파일 (기계어로 번역) arduino-cli compile --fqbn arduino:avr:uno blink # 아두이노에 업로드! arduino-cli upload -p /dev/ttyUSB0 --fqbn arduino:avr:uno blink
